## About
sjq is lint config generator for reducing jQuery.

## Install

```shell
$ npm install -g sjq
```

## Usage

When you generate config in your project

```shell
$ sjq
```


When you clone repository which includes config generated by sjq

**npm**

```shell
$ npm run sjq:prepare
```

**yarn**

```shell
$ yarn run sjq:prepare
```
